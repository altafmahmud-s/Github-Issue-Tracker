1️ What is the difference between var, let, and const?

<!-- Q-Answer 01 -->

var : can be redeclared, value can be changed (old method),
let : cannot be redeclared, value can be changed,
const : cannot be redeclared, value can't be changed

2️ What is the spread operator (...)?

<!-- Q-Answer 02 -->

The spread operator is used to extract, copy, merge, or expand elements of an array or object.

3️ What is the difference between map(), filter(), and forEach()?

<!-- Q-Answer 03 -->

map(): creates a new array,

filter(): selects elements based on a condition,

forEach(): just loops, doesn't return anything

4️ What is an arrow function?

<!-- Q-Answer 04 -->

Arrow functions are short/modern function syntax that makes code shorter and cleaner.

5️ What are template literals?

<!-- Q-Answer 05 -->

Template literals are used to place variables within a string. They are written using backtick (``). Variables can be placed inside ${}.
